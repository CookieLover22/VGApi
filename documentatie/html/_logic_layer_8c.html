int API_perform ( Q_INFO *  performQ)

Deze functie leest de meegegeven queue uit en stuurt de commando's door naar de functionpicker.

Note
Deze functie hoort in de main samen met de errorhandler.
Parameters
performQDit is een pointer naar de Q_info struct van de queue tussen de front- en logiclayer.
Returns
De functie returnt de errorcodes die door de aangeroepen functies zijn teruggegeven.

COMMAND *  writeCommand 
)

Deze functie schrijft een nieuw lid in de Q en returnt een errorcode.

Warning
Deze functie kan niet in de Queue schrijven als de Queue vol is.
Parameters
writeQDit is het Q_INFO struct waar de informatie over de Queue in staat
writeCommandDit is het COMMAND struct dat in de Queue geplaatst moet worden.
Returns
De functie returnt een errorcode, namelijk: NOERROR of FULLQ
